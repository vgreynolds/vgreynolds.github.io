ArcGIS Pro - Site Suitability

Problem: Where is the most suitable site for a new coffee shop in Sacramento, CA?
Solution: Geospatial analysis tools including network analysis, service area analysis, and kernel density were applied to data to generate a Site Suitability Analysis.
Effect: Results from the analysis indicated the Downtown location was preferred given the parameters for the analysis. A presentation was created in AGOL StoryMaps to show the results.
